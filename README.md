# calendar

# Overview
コンソール上で動くカレンダー表示のアプリケーションです。入力された年月のカレンダーをコンソール上に表示します。

# Usage
```bash
git clone https://github.com/sugurukoike/calendar.git
cd calendar
./calendar
```
